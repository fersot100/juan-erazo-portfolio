Projects

Dungeon Master Tools

DM Tools is a cross-platform electron application that runs as a window locally on your machine and helps you, the Dungeon Master, manage your role playing sessions by removing a lot of the encumberments that come with pen traditional pen and paper approaches.

I play a lot of DnD and I got frustrated having to do all my combat calculations on paper. When my players went to a place I hadn't written about, I wasn't very good at generating content on the spot. This is my attempt at putting everything you need to run a DnD campaign in your laptop screen in a user friendly way.

This app is built in React.js using Redux for state control, SCSS for styling, and runs locally on electron.

Make It Permament

Make it Permament is the result of a hyperfixation wtih base conversions. The site hosts two games that allows users to develop a proficiency converting different number bases (i.e. binary, octal, decimal, and hexadecimal) to each other and a morse code trainer.

I originally built the site when I was taking 64-bit assembly in college. Manipulating hex, octal, and binary numbers was challenging for me so I wrote this tool in C++ that could help me practice. Later I wanted to make it accessible to my peers so I converted the code to javascript and bought the domain.

This site is built with vanilla Javascript and Materialize CSS for the styling. I wanted to try something other than bootstrap.

Mom's Basement

Mom's Basement is a virtual reality experience where players must race against the clock to complete goals in a virtual space using motion controls.

I worked on this project with my friends at UCSC Hacks and it was my first time ever working virtual reality.

We built the game in Unity 3D and I made all the 3D models in blender in a 36 hour window. I has to learn blender a few weeks before we competed because we didn't have any other modelers on the team.

Ball it Up

Ball it up is just a site I made while experimenting with the HTML5 Canvas.

I wanted to make something visual from scratch and with OOP methodology. It was my first time writing collisions and kinematics from scratch without a physics engine. I learned a lot and hope to work with the canvas more.

The site is built in vanilla js and HTML5.
